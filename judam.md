# 주담(酒談) 프로젝트 AI 메모리 파일 (Frontend Focus)

이 파일은 기획안 및 사용자 요청 사항을 기반으로 프로젝트의 모든 화면별 상세 기능을 기록합니다. **프론트엔드 개발 전용** 가이드라인입니다.
이 내용을 통해 과거에 진행한 내용과 앞으로의 진행상황에 대해 바로 파악할 수 있도록 합니다.
앞으로의 모든 개발 부분에서 주석은 절대 달지 말아야 한다. 추가로, 이 파일을 본 Gemini Cli은 프론트엔드를 기준으로만 개발하며, 백엔드나 AI부분은 건들지 말아야 한다.

## 1. 프로젝트 개요
- **서비스명**: 주담(酒談) - 소비자 맞춤형 전통주 공동 기획 펀딩 플랫폼
- **디자인 컨셉**: 블랙/그레이/화이트 중심의 깔끔하고 모던한 느낌, 전통적인 요소를 세련되게 배치.

---

## 2. 화면별 상세 기능 명세 (UI/UX)

### [홈 (Home)]
- **헤더/상단 바**: 
  - AI 챗봇: 카톡 스타일 토글 UI. 우측 하단 '+' 버튼 클릭 시 위로 리스트 노출. 첫 질문 기반 제목 자동 설정.
  - 알림: 사용자용은 제거, 양조장용은 유지 (관심 수치 달성, Q&A 등록, 펀딩 목표 도달 시 알림).
  - 설정: 상단 헤더바 설정 버튼 제거.
- **양조장 대시보드 (전용)**:
  - '+ 새 펀딩' 유지, 내 펀딩 현황 하단 배치.
  - '양조장 정보 수정' 버튼: 클릭 시 펀딩 페이지의 양조장 정보 입력 폼 불러오기.
  - 펀딩 상태 토글: '진행 중인 펀딩 / 종료된 펀딩' 선택 기능.
  - 제조 진행 현황: 가로 스크롤 카드(카드 2개 노출, 하단 '..' 표시). '단계 상승' 원클릭 모달 기능.
- **메인 섹션 (순서)**:
  1. 배너 (유지)
  2. 주담이 만들어지는 과정 (가로 그리드)
  3. 현재 인기 펀딩 (펀딩 게시글 스타일 카드 3개, 전체보기 유지)
  4. 인기 레시피 제안 (주담 리스트 카드 UI 적용)
  5. 주담과 함께한 순간들 (펀딩 페이지 '주담의 성장' 디자인 개선 버전)

### [주담 (레시피 제안)]
- **UI 구조**: 전체적으로 펀딩 게시글 목록 스타일 차용.
- **헤더**: 알람, 스크랩, 검색 버튼 제거.
- **레시피 카드**: 
  - 분류(탁주 등) 삭제, 지향하는 맛은 #해시태그로 변경.
  - 댓글 아이콘 삭제, '좋아요' 아이콘을 프로필 우측에 배치.
  - 댓글 '더보기' 기능으로 확장 가능하게 구현.
- **연동**: 양조장이 '펀딩 제안하기' 클릭 시, 해당 레시피 데이터가 채워진 '펀딩 프로젝트 등록' 페이지로 이동.
- **레시피 작성 폼**:
  - 카테고리 삭제. 메인/서브 재료 입력 후 '생성' 버튼으로 리스트화.
  - AI 기능: 재료 입력 후 '생성' 클릭 시 AI가 맛 해시태그 및 프로젝트 요약(수정 가능) 생성.
  - 이미지: 사용자 업로드 + AI 메인 컨셉 기반 이미지 생성 버튼.
  - 제출: 모달을 통해 소상공인 인증 여부에 따른 버튼 노출 (미인증 시 안내 문구 포함).

### [펀딩 (Funding)]
- **프로젝트 등록**: 
  - 약관 요약(자세히 보기 포함), 숫자 가이드 제거.
  - 기본 정보: 카테고리(글 입력), 제목, 요약, 재료, 도수 입력.
  - 대표 이미지: AI 사진 생성 버튼 포함. 추가된 이미지는 우측으로 쌓임.
  - 목표 금액: (병당 단가 * 수량) 자동 계산, 수수료 7% 표시.
  - 법적 고시 정보: '리워드 구성' 대체. 상품 분류(막걸리 고정), 용량/도수 고정. 주소재/원산지는 구성요소 추가 가능.
  - 맛 지표: 단맛, 잔향, 산미, 바디감, 탄산감만 유지.
  - 양조장 정보: '불러오기' 버튼으로 대시보드 데이터 연동.
- **펀딩 상세 페이지**:
  - 헤더(하트/공유) 제거, 하단 하트 버튼 확대.
  - 레이아웃: 사진 -> 제목 -> 소제목 -> 달성률/금액/남은일수 -> 양조장 정보 순서.
  - 상세 내용: 프로젝트 소개(재료/도수/요약), 가격 안내, 일정, 맛 지표, 안내 사항(신뢰와 안전 대체).
  - 양조일지: '병입' 단계 중심, Q&A(커뮤니티 대체), 후기 섹션.

### [커뮤니티 (Community)]
- **UI**: 블랙/그레이/화이트 톤, 페이지네이션(1 2 3 4) 적용.
- **카테고리**: 자유게시판, 정보게시판 2종으로 단순화.
- **헤더**: 알림, 스크랩, 검색 제거.
- **게시글**: 상세 페이지 상단에 게시판 구분 표시(#자유/정보), 해시태그 제거, 댓글 UI는 주담(레시피) 스타일 적용.
- **작성**: '+' 버튼 클릭 시 게시판 선택, 상단 등록 버튼 제거.

### [마이페이지 (My)]
- **설정**: 프로필 사진 설정 추가. (이메일/전화번호/알림 설정 제거).
- **술BTI**: 유지.
- **목록**: 참여 펀딩/내 아카이브 리스트를 펀딩 페이지 카드 스타일로 통일.
- **경험한 술**: 펀딩 술(화이트 배경/블랙 글씨), 별점 및 마신 날짜 포함.
- **UI 컨셉**: 전반적으로 블랙 테마 적용.
