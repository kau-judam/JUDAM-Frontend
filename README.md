# JUDAM-Frontend

## 프로젝트 명: 주담 (다시 피어나는 우리 술 이야기)

### 프로젝트 소개

- **주제**: 소비자 맞춤형 전통주 공동 기획 펀딩 플랫폼

- **제안배경**: 전통주시장의 변화 / 규제 완화와 인프라 구축 / 모디슈머 성향 & 팬슈머 트렌드 확산

---

### 팀 소개

- **팀명**: 2팀 (칭찬해주)
- **팀원**:
  - 권아영
  - 강민재
  - 유승빈
  - 장요한
  - 양재원
  - 황주원

- **빌드 엔진**: Android Studio
